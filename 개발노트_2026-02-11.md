# 💇 HairApp 개발 노트 — 2026-02-11

## 프로젝트 개요
- **목적**: 미용실 고객이 QR코드 접속 → 자기 사진에 헤어스타일 AI 합성
- **기술 스택**: Next.js 15 + TypeScript + Zustand + Vanilla CSS
- **접속 경로**: `http://localhost:3000/salon/demo`

---

## Phase 1 완료 — 고객용 페이지

### 플로우
```
스플래시(2초) → 메인(사진업로드 + 스타일선택 + 컬러) → 결과(합성 + 저장/리셋)
```

### 완성된 파일

| 파일 | 역할 |
|------|------|
| `src/app/globals.css` | 디자인 시스템 (Pretendard, CSS 변수) |
| `src/app/layout.tsx` | 루트 레이아웃 (한국어, 모바일) |
| `src/app/salon/[salonId]/page.tsx` | 고객 메인 페이지 (동적 라우트) |
| `src/components/Splash/Splash.tsx` | 스플래시 (2초 자동전환, 로딩바) |
| `src/components/MainView/MainView.tsx` | 사진업로드 + 스타일갤러리 + 컬러팔레트 |
| `src/components/ResultView/ResultView.tsx` | 합성결과 + 스토리카드 + 저장 |
| `src/store/useAppStore.ts` | Zustand 전역 상태 |
| `src/data/demo.ts` | 데모 데이터 (미용실, 스타일 8종) |

### 빌드 결과
- ✅ `npm run build` 성공 (에러 0개)
- `/salon/[salonId]` 동적 라우트 108 kB
